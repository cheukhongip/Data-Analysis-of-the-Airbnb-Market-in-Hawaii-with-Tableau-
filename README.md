# Airbnb Analysis in Tableau


### Project Description
We analyzed Airbnb data in Hawaii and made a dashboard in Tableau.

### Project Coding/Softwares

* Python
* Tableau

We used listings.csv.gz and reviews.csv.gz on Sep 22, 2021, and Jan 14, 2021 in Python.
